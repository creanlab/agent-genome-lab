Replay Gate

No pattern becomes a rule without proof. The system tests: does this fix work on other similar failures? Only ≥60% success rate gets promoted.

Admission Gate

Skills pass through quarantine → evaluation → admission. 5-axis scoring ensures only quality patterns enter your agent's memory.

Utility Decay

Knowledge degrades without use. 30 days idle → score ×0.9. 90 days → ×0.7. Each reuse → ×1.1, each prevention → ×1.2. Memory stays fresh and relevant.

Toolkit

16 CLI tools, zero dependencies

Every tool is a standalone Node.js script. No npm install, no Docker, no API keys. Just node cli/tool.js.

Core Pipeline (12 tools)

nve-init · nve-scaffold · nve-memory · nve-distill · nve-replay · nve-audit · nve-validate · nve-utility · nve-pack · nve-fg-summary · nve-manifest · nve-export-dashboard

SkillGraph Pipeline (4 tools)

nve-skill-extract · nve-skill-index · nve-skill-package · nve-skill-search

NEW in v2.3.0

# Full pipeline in 8 commands:
node cli/nve-distill.js                        # incidents → EU → FG
node cli/nve-replay.js                         # replay gate (promote/reject)
node cli/nve-skill-extract.js                  # genomes → candidate skills
node cli/nve-skill-index.js                    # evaluate + dedupe + relations
node cli/nve-skill-package.js --auto --publish # build packages + publish
node cli/nve-memory.js                         # regenerate MEMORY.md
node cli/nve-audit.js                          # 5-axis + SkillGraph score
node cli/nve-skill-search.js "query"           # search skills locally
